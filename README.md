# PPG心率数据采集系统 (STM32L452CEU6)

基于STM32L452CEU6的多传感器数据采集系统，支持PPG心率/血氧、MIMU运动数据和界面热传感器数据的同步采集与蓝牙传输。

## 项目概述

本项目实现了一个完整的多通道生理信号采集系统，主要功能包括：

- **PPG信号采集**: 支持心率和血氧两种工作模式
- **MIMU数据采集**: 加速度计、陀螺仪、磁力计数据
- **界面热传感器数据采集**: 4通道ADC数据采集
- **蓝牙数据传输**: 115200波特率无线传输
- **温度补偿**: 支持芯片温度监测与补偿

## 硬件配置

### 主控芯片
- **MCU**: STM32L452CEU6
- **主频**: 80MHz (通过PLL从HSE倍频)
- **供电**: 5.0V和1.8V双电源轨

### 传感器模块

#### 1. MAX30101 PPG传感器 (I2C1接口)
- **接口**: I2C1 (PA0=SCL, PA1=SDA)
- **功能**: 光电容积脉搏波测量
- **支持模式**:
  - 心率模式: 绿光LED (660mA)
  - 血氧模式: 红光+红外LED (各29mA)
- **采样率**: 可配置 (50-3200 SPS)

#### 2. ADS124S06 ADC (SPI1接口)
- **接口**: SPI1 (PA4=NSS)
- **功能**: 高精度ADC数据采集
- **采集通道**: AIN0~AIN3 (4通道)
- **精度**: 24-bit

#### 3. LSM9DS1 MIMU (SPI2接口)
- **接口**: SPI2 (PB7=CS_A_G, PB8=CS_M)
- **功能**: 9轴运动传感器
  - 加速度计: 3轴
  - 陀螺仪: 3轴
  - 磁力计: 3轴
- **中断**: PB3(INT2_A_G), PB4(INT1_A_G), PB5(INT2_M), PB6(DRDY_M)

#### 4. BLE蓝牙模块
- **接口**: UART2 (默认19200, 配置后115200)
- **控制引脚**: PB12(STATE), PB13(RST)
- **功能**: 无线数据传输

### GPIO引脚映射

| 功能 | 引脚 | 描述 |
|------|------|------|
| I2C1_SCL | PA0 | MAX30101时钟线 |
| I2C1_SDA | PA1 | MAX30101数据线 |
| SPI1_NSS | PA4 | ADS124片选 |
| V5_0_CE | PA8 | 5.0V电源使能 |
| V1_8_CE | PA9 | 1.8V电源使能 |
| START_CONV | PA10 | ADC启动转换 |
| DRDY | PA11 | ADC数据就绪 |
| AD_RESET | PA12 | ADC复位 |
| DEN_A_G | PA15 | MIMU使能 |
| I2C2_SCL | PB0 | I2C2时钟线 |
| I2C2_SDA | PB1 | I2C2数据线 |
| INT2_A_G | PB3 | MIMU中断2 |
| INT1_A_G | PB4 | MIMU中断1 |
| INT2_M | PB5 | 磁力计中断2 |
| DRDY_M | PB6 | 磁力计数据就绪 |
| CS_A_G | PB7 | 加速度计/陀螺仪片选 |
| CS_M | PB8 | 磁力计片选 |
| BLE_STATE | PB12 | 蓝牙状态指示 |
| BLE_RST | PB13 | 蓝牙复位信号 |

## 工作模式

### 模式切换

通过修改 `main.h` 中的宏定义来切换工作模式：

```c
// 心率模式
#define CURRENT_WORK_MODE MODE_HEART_RATE

// 血氧模式
#define CURRENT_WORK_MODE MODE_SPO2
```

**心率模式特点**:
- 使用绿光LED (LED3)
- Multi-LED模式配置
- 4倍硬件平均
- 1000 SPS内部采样率

**血氧模式特点**:
- 使用红光+红外LED (LED1 + LED2)
- 400 SPS采样率
- 支持18-bit ADC分辨率
- 温度补偿功能

## 数据格式

### 数据包结构 (21字节)

| 字节索引 | 内容 | 描述 |
|----------|------|------|
| 0 | 0xAA | 帧头1 |
| 1 | 0xBB | 帧头2 |
| 2-9 | ADC数据 | 4通道ADC数据 (每通道2字节) |
| 10-12 | ACC数据 | 加速度计高字节 (X, Y, Z) |
| 13-16 | PPG数据 | PPG数据 (心率模式或血氧模式) |
| 17-18 | 扩展数据 | 温度数据或模式标志 |
| 19 | 校验位 | XOR校验 (字节2-18) |
| 20 | 0xCC | 帧尾 |

### 心率模式PPG数据格式
- 字节13-15: 绿光PPG数据 (24-bit)
- 字节16: 采样计数
- 字节17: 0x00 (补零)
- 字节18: 0xFF (心率模式标志)

### 血氧模式PPG数据格式
- 字节13-14: 红光PPG数据 (16-bit, 右移2位)
- 字节15-16: 红外PPG数据 (16-bit, 右移2位)
- 字节17: 温度整数部分
- 字节18: 温度小数部分

## 编译与烧录

### 编译步骤

1. 打开Keil MDK工程文件: `MDK-ARM/L452CEU6/L452CEU6.uvprojx`
2. 选择目标: `L452CEU6`
3. 点击编译或按F7

### 烧录步骤

1. 连接ST-Link调试器
2. 点击下载或按F8
3. 烧录文件位于: `MDK-ARM/L452CEU6/L452CEU6.hex`

## 蓝牙配置

蓝牙模块初始化流程 (代码中已注释，需要时取消注释):

1. 复位蓝牙模块
2. 以默认19200波特率发送配置命令:
   - 设置为常开模式
   - 设置发射功率为+2.5dBm
   - 设置蓝牙名称为"HJ-131-LYX-5"
   - 设置配对密码为"123456"
   - 切换波特率至115200
3. MCU切换至115200波特率
4. 继续发送后续配置命令

## 系统时序

```
125Hz定时器中断 (TIM16)
    |
    +-- 触发ADC转换 (START_CONV)
    |
    +-- DRDY中断 (GPIO)
        |
        +-- 读取ADC数据 (4通道)
        +-- 读取MIMU数据 (ACC)
        +-- 主循环打包数据
        +-- DMA发送21字节数据包
```

## 调试信息

系统通过UART2 (115200波特率) 输出调试信息：

## 注意事项

1. **电源时序**: 必须在SPI初始化之后开启传感器电源，避免引脚干扰
2. **波特率切换**: 蓝牙模块配置时需要先发送波特率切换指令，然后MCU再切换波特率
3. **FIFO管理**: MAX30101的FIFO需要在每次读取后排空
4. **模式切换**: 修改工作模式后需要重新编译烧录
5. **校验位**: 数据包使用XOR校验，校验范围为字节2-18 (17字节)
