# ADS124S06 桥压 ADC 配置说明

> 基于 ADS124S06 数据手册和本项目 `Core/Src/ADC_ADS124.c` 实际配置整理。

---

## 1. 硬件接口

| 项目 | 说明 |
|------|------|
| 通信总线 | SPI1 (SCK: PA5, MISO: PA6, MOSI: PA7) |
| 片选引脚 | SPI1_NSS = PA4 (低电平有效) |
| 启动转换 | START_CONV (GPIO 输出) |
| 复位引脚 | AD_RESET (GPIO 输出) |
| 数据就绪 | DRDY (EXTI 下降沿触发) |
| 参考电压 | 内部 2.5V, 常开 |

### 硬件连接

热膜桥压传感器通过 4 路 AIN 通道接入 ADC:
- AIN0, AIN1: 桥顶电压 (Ut1, Ut2)
- AIN2, AIN3: 桥中电压 (Uc1, Uc2)
- AINCOM: 公共负端 (飞线接 2.5V)
- AIN5: 接地 (MUXN 备选)

## 2. 寄存器配置

### 2.1 初始化序列 (`ADC_Init`)

```
Step 1: 硬件复位     -> AD_RESET 拉低 1ms -> 拉高 -> 等待 10ms
Step 2: 等待就绪     -> 轮询 STATUS[6]=0, 最多重试10次
Step 3: 清除POR标志  -> STATUS (0x01) = 0x00
Step 4: MUX配置      -> MUX (0x02) = 0x3C (AIN3 / AINCOM)
Step 5: PGA关闭      -> GSET (0x03) = 0x00 (PGA bypass)
Step 6: 采样率       -> RATE (0x04) = 0x2C (2000sps, low-latency)
Step 7: 参考电压     -> RCON (0x05) = 0x3A (内部2.5V, 常开)
Step 8: 偏置电压     -> BIAS (0x08) = 0x00 (不连接)
Step 9: 系统控制     -> SCON (0x09) = 0x10 (关闭测试模式)
```

### 2.2 各寄存器详解

#### STATUS (0x01) - 状态寄存器

| Bit | 名称 | 说明 |
|-----|------|------|
| 7 | FL_POR | 上电复位标志, 写0清除 |
| 6 | RDY | 0=数据就绪, 1=正在转换 |

#### MUX (0x02) - 输入多路复用器

| Bit | 名称 | 说明 |
|-----|------|------|
| [7:4] | MUXP | 正端输入选择 |
| [3:0] | MUXN | 负端输入选择 (0xC = AINCOM) |

本项目通过 `ADC_1to4Voltage()` 状态机轮转切换通道:
- `0x0C`: AIN0 / AINCOM
- `0x1C`: AIN1 / AINCOM
- `0x2C`: AIN2 / AINCOM
- `0x3C`: AIN3 / AINCOM

#### GSET (0x03) - PGA 与增益配置

| Bit | 名称 | 值 | 说明 |
|-----|------|---|------|
| [2:0] | GAIN | 0 | PGA bypass, 增益=1 |
| 3 | PGA_EN | 0 | PGA 禁用 |

本项目不使用 PGA, 直接以增益=1 读取桥压。

#### RATE (0x04) - 数据速率

本项目配置 = `0x2C`:
- 滤波器: Low-latency 模式
- 数据速率: 2000 sps

#### RCON (0x05) - 参考电压控制

本项目配置 = `0x3A`:
- 参考源: 内部 2.5V
- 参考始终开启 (降低首次转换延迟)

#### SCON (0x09) - 系统控制

本项目配置 = `0x10`:
- 关闭内部测试模式
- CRC 等功能未启用

## 3. 数据采集流程

### 3.1 四通道轮转 (状态机)

`ADC_1to4Voltage()` 在主循环中被调用, 每次切换一个通道并启动转换:

```
flag=0: MUX=AIN0/AINCOM -> START拉高 -> flag=1
flag=1: MUX=AIN1/AINCOM -> START拉高 -> flag=2
flag=2: MUX=AIN2/AINCOM -> START拉高 -> flag=3
flag=3: MUX=AIN3/AINCOM -> START拉高 -> flag=0
```

### 3.2 DRDY 中断读取

DRDY 下降沿触发 EXTI 中断, 在 `HAL_GPIO_EXTI_Callback` 中调用 `ADC_RDATA()`:

```c
// 发送 4 字节: DIN[0] 为 RDATA 命令 (0x12), 后3字节为空
// 接收 4 字节: 第1字节为废位, 后3字节中取前2字节为24位ADC值的高16位
HAL_SPI_TransmitReceive(&hspi1, DIN, pRxBuf, 4, 0xffff);
```

实际只取前 2 字节 (16-bit), 舍弃最低字节, 作为毫伏级电压值打包上传。

---

## 4. 数据协议映射

在 33 字节原始数据包中, ADC 数据占据以下字段:

| 偏移 | 长度 | 字段 | 说明 |
|------|------|------|------|
| 2~3 | 2B | Uc1 (mV) | 桥中电压 1 |
| 4~5 | 2B | Uc2 (mV) | 桥中电压 2 |
| 6~7 | 2B | Ut1 (mV) | 桥顶电压 1 |
| 8~9 | 2B | Ut2 (mV) | 桥顶电压 2 |

## 5. 寄存器速查表

| 地址 | 名称 | 本项目值 | 说明 |
|------|------|---------|------|
| 0x00 | ID | 只读 | 设备ID (bit0=1表示ADS124S06) |
| 0x01 | STATUS | 0x00 | 清除POR标志 |
| 0x02 | MUX | 0x0C~0x3C | 四通道轮转 |
| 0x03 | GSET | 0x00 | PGA bypass |
| 0x04 | RATE | 0x2C | 2000sps, low-latency |
| 0x05 | RCON | 0x3A | 内部2.5V参考 |
| 0x06 | CUR1 | 默认 | IDAC1未使用 |
| 0x07 | CUR2 | 默认 | IDAC2未使用 |
| 0x08 | BIAS | 0x00 | 不连接偏置电压 |
| 0x09 | SCON | 0x10 | 关闭测试模式 |
