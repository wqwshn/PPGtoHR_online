# LSM9DS1 六轴 IMU 传感器配置说明

> 基于 LSM9DS1 数据手册和本项目 `Core/Src/MIMU.c` 实际配置整理, 供开发调试时快速查阅。

---

## 1. 硬件接口

| 项目 | 说明 |
|------|------|
| 通信总线 | SPI2 (SCK: PB13, MISO: PB14, MOSI: PB15) |
| ACC/GYRO 片选 | CS_A/G = PB7 (低电平有效) |
| 磁力计片选 | CS_M = PB8 (低电平有效) |
| SPI 模式 | CPOL=1, CPHA=1 (Mode 3) |
| SPI 最大时钟 | 10 MHz |
| WHO_AM_I (0x0F) | A/G 返回 `0x68`, 磁力计返回 `0x3D` |

### 寄存器寻址机制

- A/G 传感器命令字节最高位 (Bit 7) 为 R/W: 0=写, 1=读
- 开启 `IF_ADD_INC` 后, 突发读取地址自动递增

## 2. 本项目实际寄存器配置

### 2.1 初始化序列 (`MIMU_Init`)

```
Step 1: CTRL_REG8 (0x22) = 0x05    // 软复位 + IF_ADD_INC
        延时 20ms

Step 2: CTRL_REG8 (0x22) = 0x44    // BDU=1 + IF_ADD_INC=1

Step 3: CTRL_REG9 (0x23) = 0x04    // 禁用 FIFO, 禁用 I2C

Step 4: CTRL_REG1_G (0x10) = 0x68  // 陀螺仪: 119Hz, +/-500dps, BW=14Hz
        CTRL_REG3_G (0x12) = 0x46  // 陀螺仪: 高通滤波使能, 截止约0.1Hz

Step 5: CTRL_REG6_XL (0x20) = 0x70 // 加速度计: 119Hz, +/-4g, 自动带宽
        CTRL_REG7_XL (0x21) = 0x00 // 加速度计: 无额外滤波

Step 6: 磁力计配置 (CTRL_REG1~5_M)
```

### 2.2 各寄存器位域详解

#### CTRL_REG8 (0x22) - 系统控制

| Bit | 名称 | 本项目配置 | 说明 |
|-----|------|-----------|------|
| 7 | BOOT | 0 | 不重启内存 |
| 6 | BDU | 1 | **防撕裂**: 高低字节成对读取后才更新 |
| 5 | H_LACTIVE | 0 | 中断高电平有效 |
| 4 | PP_OD | 0 | 推挽输出 |
| 3 | SIM | 0 | 4线SPI |
| 2 | IF_ADD_INC | 1 | **地址自增**: 突发读取必备 |
| 1 | BLE | 0 | 小端序 |
| 0 | SW_RESET | 1->0 | 初始化时触发软复位 |

初始化分两步: 先写 `0x05` 触发复位, 延时 20ms 后写 `0x44` 进入工作态。

#### CTRL_REG9 (0x23) - FIFO/I2C 控制

| Bit | 名称 | 本项目配置 | 说明 |
|-----|------|-----------|------|
| 4 | FIFO_TEMP_EN | 0 | 温度不存入FIFO |
| 2 | I2C_DISABLE | 1 | **禁用I2C**, 仅使用SPI |
| 1 | FIFO_EN | 0 | **禁用FIFO**, 主动轮询读取 |

本项目配置 = `0x04`, 彻底旁路 FIFO, 由 MCU 定时器主动发起 SPI 读取。

#### CTRL_REG1_G (0x10) - 陀螺仪基础配置

| Bit | 名称 | 值 | 说明 |
|-----|------|---|------|
| [7:5] | ODR_G | 011 | 119 Hz |
| [4:3] | FS_G | 01 | +/-500 dps |
| [1:0] | BW_G | 00 | 14 Hz 带宽 |

本项目配置 = `0x68`

#### CTRL_REG3_G (0x12) - 陀螺仪滤波配置

| Bit | 名称 | 值 | 说明 |
|-----|------|---|------|
| 6 | HP_EN | 1 | 高通滤波使能, 消除零偏漂移 |
| [3:0] | HPCF_G | 0110 | 截止频率约 0.1Hz (ODR=119Hz) |

本项目配置 = `0x46`

#### CTRL_REG6_XL (0x20) - 加速度计基础配置

| Bit | 名称 | 值 | 说明 |
|-----|------|---|------|
| [7:5] | ODR_XL | 011 | 119 Hz |
| [4:3] | FS_XL | 10 | +/-4 g |
| [2] | BW_SCAL_ODR | 0 | 带宽自动由ODR决定 |
| [1:0] | BW_XL | 00 | 防混叠滤波带宽 (ODR/2) |

本项目配置 = `0x70`

#### CTRL_REG7_XL (0x21) - 加速度计高级配置

| Bit | 名称 | 值 | 说明 |
|-----|------|---|------|
| 7 | HR | 0 | 普通模式 |
| [6:5] | DCF | 00 | 滤波截止频率 (未启用) |
| 2 | FDS | 0 | **不**将滤波后数据送输出寄存器 |

**本项目配置 = `0x00`**

> **重要踩坑记录**: 参考项目使用 `CTRL_REG7_XL = 0xC4` (HR=1, DCF=ODR/9, FDS=1)。
> 实测发现 FDS=1 会启用 LPF2 数字滤波器, 导致 ACC 输出趋近于零。
> 本项目硬件上 ACC 输出直接使用未经 LPF2 的原始数据, 因此 FDS 必须为 0。
> 详见 `bug/ACC三轴归零问题排查记录.md`。

## 3. 磁力计配置 (LSM9DS1 内部)

本项目启用了磁力计, 配置如下:

| 寄存器 | 地址 | 值 | 说明 |
|--------|------|-----|------|
| CTRL_REG1_M | 0x20 | 0xFC | 温度补偿, XY轴UHP模式, ODR=80Hz |
| CTRL_REG2_M | 0x21 | 0x00 | 量程 +/-4 gauss |
| CTRL_REG3_M | 0x22 | 0x80 | 禁用I2C, SPI模式, 连续转换 |
| CTRL_REG4_M | 0x23 | 0x0C | Z轴UHP模式 |
| CTRL_REG5_M | 0x24 | 0x00 | 连续转换模式 |

## 4. 数据读取方式

### 4.1 突发读取 (6字节)

本项目对 ACC 和 GYRO 分别进行 6 字节突发读取, 而非合并为 12 字节:

```
ACC:  起始地址 OUT_X_L_XL (0x28), 读取 7 字节 (1命令 + 6数据)
GYRO: 起始地址 OUT_X_L_G  (0x18), 读取 7 字节 (1命令 + 6数据)
```

读取缓冲区为 `ACC_XYZ[6]` 和 `GYRO_XYZ[6]`, 数据为小端序 (低字节在前)。

### 4.2 物理量转换

| 传感器 | 量程 | 灵敏度 | 换算公式 |
|--------|------|--------|---------|
| ACC | +/-4g | 0.122 mg/LSB | `acc_g = raw * 0.122 / 1000` |
| GYRO | +/-500dps | 17.50 mdps/LSB | `gyro_dps = raw * 17.50 / 1000` |

### 4.3 Z轴方向修正

PCB 上传感器 Z 轴正方向朝下, 因此在数据打包时对 ACC_Z 取反:

```c
int16_t az = -(int16_t)((ACC_XYZ[5] << 8) | ACC_XYZ[4]);
```

## 5. 陀螺仪零偏标定

### 5.1 标定流程

标定函数 `MIMU_GyroCalibrate()` 在每次上电时执行, 分四个阶段:

| 阶段 | 样本数 | 耗时 | 说明 |
|------|--------|------|------|
| 暖机丢弃 | 200 | ~2s | 丢弃传感器刚启动时的不稳定数据 |
| 有效采集 | 800 | ~8s | 累加三轴原始值, 每200样本输出进度 |
| 标准差校验 | 200 | ~2s | 验证静置状态, std > 3.0 dps 则拒绝 |
| Flash存储 | - | <1ms | 写入最后一页 (0x0807F800) |

### 5.2 Flash 存储格式

```c
typedef struct __attribute__((packed)) {
    uint32_t magic;       // 0xAA55CC77
    int16_t  offset[3];   // X/Y/Z 零偏 (LSB)
    uint8_t  reserved;    // 0xFF
    uint8_t  xor_check;   // 异或校验
} GyroCalibFlash_t;
```

存储位置: Flash 最后一页 (0x0807F800), 占 16 字节 (2个双字)。

### 5.3 开机自动加载

`MIMU_LoadGyroOffset()` 在 `MIMU_Init()` 之后调用, 校验 magic 和 xor 后加载零偏。
即使标定被拒绝 (std 过高), 也不会清除 Flash 中已有的有效零偏。

## 6. 与参考项目配置对比

| 项目 | 本项目 | 参考项目 (another-referrence) |
|------|--------|------|------|
| SPI 总线 | SPI2 | SPI1 |
| CS_A/G 引脚 | PB7 | PD7 |
| ACC 量程 | +/-4g | +/-2g |
| CTRL_REG7_XL | **0x00** (无LPF2) | 0xC4 (HR+DCF+FDS) |
| CTRL_REG9 | 0x04 (禁I2C) | 0x00 |
| 读取方式 | 分离6字节读取 | 12字节DMA合并读取 |
| 磁力计 | 已启用 | 未启用 |
| 陀螺仪标定 | 暖机+800样本+std校验+Flash | 无 |
| 数据处理 | 软件零偏减除 + Z轴取反 | 硬件高通滤波 |
